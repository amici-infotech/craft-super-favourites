# Changelog

All notable changes to this project will be documented in this file.

## 5.0.0 - 2024-11-17

### Added
- Initial release for Craft CMS 5.x (compatible with 4.x and 5.x)
- Custom Collection element for organizing favourites into lists
- Custom FavouriteItem element for tracking favourited items
- Support for favouriting any Craft element type (Entries, Assets, Categories, Users, etc.)
- User authentication requirement for favouriting
- Frontend forms for adding and removing favourites
- Template variables for querying favourites
- Control Panel interface for managing collections
- PHP API for programmatic access
- User permissions system
- Settings page with customizable options
- Default collection auto-creation
- Multiple collection support per user
- Comprehensive database architecture with proper indexes and foreign keys
- Full documentation with usage examples
